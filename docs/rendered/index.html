Safety First

MechWolf's compiler has your back. Design, simulate, and analyze experiments before connecting to real hardware to make sure they'll work.

Simplicity

Follow the same process as in the lab, except now in Python: design an apparatus, create a protocol, execute it, and analyze the results.

Interactive

Use an interactive browser-based notebook to visualize your apparatus and protocol and then monitor the experiment in real time with live graphs.

# MechWolf doesn't bite

# Python 3.7 or above required!
$ python -V
Python 3.7.3

# install
$ pip3 install mechwolf

# open an interactive notebook
$ jupyter notebook
